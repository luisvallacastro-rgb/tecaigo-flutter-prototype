# TeCaiGO Flutter Native

Base inicial para convertir el prototipo web de TeCaiGO en una app nativa Flutter.

## Pantallas incluidas

- Login nativo con credenciales iniciales.
- HomeFeed movil con filtros por interes turistico.
- Cards sociales con fotos, comentarios, metricas y acciones.
- Navegacion inferior para Home, Crear, Solicitudes y Mas.
- Drawer con modulos futuros: validaciones, mis eventos, eventos publicos, finanzas, clusters y admin.

## Credenciales de prueba

- Usuario: `operador@tecaigo.com`
- Clave: `TecaiGO2026`

## Ejecutar

Instala Flutter y luego corre:

```bash
flutter pub get
flutter run
```

En este workspace no estaba instalado el comando `flutter`, por eso la base fue creada manualmente.
